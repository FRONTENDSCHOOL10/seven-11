## 프로젝트 후기

| 팀원 이름 | 후기 |
| :-----------------| :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 박윤선                  | 이번 프로젝트를 하면서 taiwindcss 사용에 대한 공부를 할 수 있어서 개인적으로 좋았고 리액트를 사용함에 있어서 taiwindcss가 편리하다고 느꼈습니다.비동기 데이터 처리에 어려움이 많았는데 여러가지 에러를 마주하며 통신 에러 조건 처리에 조금은 익숙해진 것 같습니다. zustand를 사용한 상태관리를 통해 zustand의 편리함을 느꼈고 다시 한 번 초반의 컴포넌트 설계가 얼마나 중요한 지 깨달았습니다. |
| 이아영               | 바닐라 때보다 훨씬 길었던 프로젝트 기간인 만큼, 프로젝트의 규모도 커지고, 부담감과 스트레스가 훨씬 더 컸던 것 같습니다. 오래 걸리더라도 초반의 데이터나 플로우 등의 설계가 얼마나 중요한지 느낄 수 있었습니다. 힘들었지만 좋은 팀원들을 만나서 제가 생각했던 제 역량보다 훨씬 많은 것을 해낼 수 있었던 것 같아 또 한 번 성장하는 시간이 되었던 것 같습니다. |
| 장민혁                | 전반적인 프론트엔드 기술들에 대한 이해도, 사용 수준 향상, GPT 마스터 달성 혼자라면 절대 완성도 못하고 기능 구현도 제대로 하지 못했을 텐데 팀원들과 함께라 이 모든 과정들을 이겨낼 수 있었습니다. |
